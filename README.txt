🚀 Customer Churn Prediction System (End-to-End ML Project)
📌 Overview

This project is a full-stack Machine Learning system that predicts customer churn risk using a trained classification model and a production-ready FastAPI backend with an interactive frontend dashboard.

The system enables businesses (Telecom / SaaS) to proactively identify high-risk customers and take retention actions.

🧠 Problem Statement

Customer churn leads to significant revenue loss.
By predicting churn probability, companies can:

Offer retention discounts

Provide targeted support

Reduce revenue leakage

Improve lifetime value
